  <action>
    Create scripts/health-check.mjs with the following behavior:
    - Accept a URL from command line: `node scripts/health-check.mjs <url>` (default URL is `http://localhost:3000/health`)
    - Use global `fetch` to GET the URL
    - Measure response time with `performance.now()` (or `Date.now()` difference)
    - Parse JSON response
    - Print a formatted line: `Health: <status> | Version: <version> | Env: <environment> | Response Time: <ms>ms`
    - If the fetch fails or JSON is invalid, print `Health check FAILED: <error.message>` and exit with code 1
    - If status is not "ok", print `Health check WARNING: status=<status>` and exit with code 1
    - On success, exit with code 0
    Use only standard Node.js APIs (fetch is available in Node 18+).
  </action>
