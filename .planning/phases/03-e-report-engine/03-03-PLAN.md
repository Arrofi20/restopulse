  <action>
    Install the PDF generation libraries and build the export engine.

    1. Install packages in the frontend directory:
       ```bash
       cd frontend && npm install jspdf@4.2.1 jspdf-autotable@5.0.8
       ```
       Per RESEARCH.md Package Legitimacy Audit, both packages are OK (verified via npm registry).

    2. Create frontend/src/lib/pdfGenerator.ts:
       - Import `jsPDF` from 'jspdf' and `autoTable` from 'jspdf-autotable'.
       - Import `formatRupiah` from './format'.
       - Import `ReportData` from '../types/report'.
       - Export function `generateReportPDF(data: ReportData): void`.
       - Create doc: `new jsPDF({ orientation: 'portrait', unit: 'mm', format: 'a4' })`.
       - **Header section** (y=20):
         - Outlet name: `doc.setFontSize(18); doc.setTextColor(20, 20, 20); doc.text(data.outlet.name, 14, 20);`
         - Period: `doc.setFontSize(11); doc.setTextColor(80, 80, 80); doc.text(\`Periode: \${data.period.start} s/d \${data.period.end}\`, 14, 28);`
       - **Summary section** (y=38):
         - `doc.setFontSize(10);`
         - `Total Omset: ${formatRupiah(data.summary.totalRevenue)}`
         - `Jumlah Transaksi: ${data.summary.transactionCount}`
         - `Menu Terlaris: ${data.summary.topItems.slice(0, 3).join(', ') || '-'}`
       - **Daily table** via `autoTable(doc, { ... })`:
         - `startY: 56`
         - `head: [['Tanggal', 'Omset (Rp)', 'Menu Terlaris', 'Jumlah Transaksi']]`
         - `body: data.rows.map(r => [r.date, formatRupiah(r.revenue), r.topMenu, String(r.transactionCount)])`
         - `theme: 'grid'`
         - `headStyles: { fillColor: [240, 240, 240], textColor: 20, fontStyle: 'bold' }` (light gray header for print readability)
         - `bodyStyles: { textColor: 20 }` (dark text on white paper per D-25b)
         - `styles: { fontSize: 10, cellPadding: 2 }`
         - `columnStyles: { 0: { cellWidth: 30 }, 1: { halign: 'right', cellWidth: 40 }, 2: { cellWidth: 'auto' }, 3: { halign: 'center', cellWidth: 35 } }`
         - `pageBreak: 'auto'`
         - `showHead: 'everyPage'`
         - `rowPageBreak: 'avoid'` (prevents rows from splitting across pages — RESEARCH.md Pitfall 3)
         - `didDrawPage: (dataArg) => { ... }` — footer text: `Dibuat pada: ${new Date().toLocaleString('id-ID')} — Halaman ${dataArg.pageNumber} / ${pageCount}` at y = pageHeight - 10.
       - **File name**: sanitize outlet name by removing/replacing non-alphanumeric chars with underscore. `doc.save(\`Laporan_\${safeName}_\${data.period.start}_\${data.period.end}.pdf\`)`.
       - Per D-25b, the PDF always uses white paper background with dark text — no dark theme in the exported file.
       - Per D-36, generation happens entirely client-side.
  </action>
