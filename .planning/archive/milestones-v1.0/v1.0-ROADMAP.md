# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-06-27
**Phases:** 1-5
**Total Plans:** 23

## Overview

RestoPulse v1.0 MVP — Sistem informasi dasbor analitik dan laporan digital (E-Report) berbasis web untuk pemilik restoran. Mengubah data penjualan harian menjadi grafik interaktif (tren omset dan menu terlaris) serta menghasilkan dokumen laporan keuangan digital yang siap unduh dan cetak.

## Phases

### Phase 1: Foundation

**Goal**: Sistem backend siap dengan database, autentikasi pemilik, API input data manual, dan generator data simulasi.
**Depends on**: Nothing (first phase)
**Plans**: 5 plans

Plans:

- [x] 01-01: Design database schema (OwnerAccount, DailySales, SalesTrend, StatusLog, DailySalesReport) dan setup Prisma dengan SQLite
- [x] 01-02: Implementasi API autentikasi (register/login, bcryptjs hashing, JWT session)
- [x] 01-03: Implementasi API input data transaksi harian manual (validasi tanggal, omset, menu terlaris)
- [x] 01-04: Implementasi dummy data injector (generate data fiktif 365 hari dengan distribusi realistis)

**Details:**

## Milestone Summary

**Key Decisions:**

- Web-based dashboard (bukan mobile app) — akses fleksibel via browser
- Manual data entry + dummy injector untuk MVP — tidak ada integrasi POS real-time
- Chart.js untuk visualisasi — matang, ringan, tooltip out-of-the-box
- Single-outlet constraint untuk v1 — menghindari kompleksitas multi-tenant
- PDF + CSV untuk ekspor laporan — format paling umum untuk arsip internal
- Render.com hosting — free-tier Web Service + PostgreSQL + Static Site
- SQLite local dev, PostgreSQL production — env-based Prisma datasource
- Prisma 7 adapter-based config — prisma-adapter-sqlite untuk local

**Issues Resolved:**

- Prisma 7 incompatibilities: env() tidak didukung di provider argument, url dihapus dari schema files, PrismaClient memerlukan adapter
- tsconfig rootDir diubah dari '.' ke './src' agar tsc menghasilkan dist/server.js
- formatCompactRupiah locale bug — dot decimal diganti comma untuk id-ID
- tsc --noEmit vs tsc -b discrepancy — tooltip null guard + verifikasi build
- 7 Major touch target issues (<44px) diaksesibilitas — semua diperbaiki

**Issues Deferred:**

- UAT Phase 03 (E-Report Engine) — 6 pending scenarios (acknowledged at close)
- Verification Phase 02 (Dashboard) — gaps_found (acknowledged at close)

- JWT payload shape validation — Minor severity, deferred to v2
- Chart.js canvas accessibility — Minor severity, deferred to v2
