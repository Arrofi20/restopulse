export interface MenuPopularityItem {

**B. `src/lib/format.ts`** — per PATTERNS.md §11 (exact code from RESEARCH.md lines 537-565):
- Create `Intl.NumberFormat('id-ID', { style: 'currency', currency: 'IDR', minimumFractionDigits: 0, maximumFractionDigits: 0 })` as a module-level singleton.
- Export `formatRupiah(amount: number): string` — returns "Rp 12.345.678".
- Export `formatCompactRupiah(amount: number): string` — returns "Rp 12,3 jt" for millions, "Rp 1,2 M" for billions, falls back to formatRupiah.
- DO NOT format negative numbers specially — revenue should not be negative.

**C. `src/lib/chartConfig.ts`** — per PATTERNS.md §12:
- Import Chart.js components: `Chart, CategoryScale, LinearScale, PointElement, LineElement, ArcElement, Title, Tooltip, Legend, Filler` from 'chart.js'.
- Call `ChartJS.register(CategoryScale, LinearScale, PointElement, LineElement, ArcElement, Title, Tooltip, Legend, Filler)` at module level.
- Export `CHART_COLORS` constant with the dark theme palette from RESEARCH.md §Open Questions #3:
  - `background: '#1a1a2e'`, `cardBg: '#16213e'`, `primaryText: '#ffffff'`, `accent: '#fbbf24'` (amber-400), `decline: '#ef4444'` (red-500), `chartGrid: 'rgba(255,255,255,0.1)'`, `tooltipBg: 'rgba(0,0,0,0.85)'`.
- DO NOT import `TimeScale` or `TimeSeriesScale` — we use category scale.
- DO NOT register `ArcElement` twice.

**D. `src/api/client.ts`** — per PATTERNS.md §8 (exact code from RESEARCH.md lines 569-630):
- `API_BASE = '/api'` (relative path — Vite proxy handles routing).
- `getToken()`: reads `localStorage.getItem('restopulse_token')`.
- `setToken(token: string)`: writes to localStorage.
- `clearToken()`: removes from localStorage.
- `apiClient<T>(endpoint, options)`: generic fetch wrapper that:
  - Attaches `Authorization: Bearer <token>` header when token exists.
  - Sets `Content-Type: application/json`.
  - On 401 response: calls `clearToken()`, redirects to `/login` via `window.location.href`, throws `'Session expired'`.
  - On non-ok response: parses error JSON (catches JSON parse failures), throws with `error.error.message` or HTTP status.
  - On success: returns `response.json()` as T.
- Export `get<T>(endpoint)` and `post<T>(endpoint, body)` convenience functions.
- DO NOT use `credentials: 'include'` — the backend uses Bearer tokens, not cookies (per RESEARCH.md Pitfall 2).
- Handle the `Authorization` header type cast: `(headers as Record<string, string>)['Authorization']` to avoid TS index errors.

Create the `api/`, `types/`, and `lib/` directories under `frontend/src/` if they don't exist.
